# PeerCritic
A website for movies and songs

## Run server
- Run: "python -m venv .venv"
- Run: "source .venv/Scripts/activate"
- Run: "pip install -r requirements.txt"

- Add secret file .env
- Run: "fastapi dev main.py"
- Go to http://127.0.0.1:8000/docs for REST API documentation
- Go to http://127.0.0.1:8000/admin for Admin Dashboard

## Run frontend
- Run: "cd frontend"
- Run: "npm install"
- Run: "npm run dev"
- Go to http://localhost:3000
